# 更新记录

## V1.2.0 - 2026-05-27

- 自选行情从仅支持 A 股扩展为可解析新浪代码或名称，覆盖常见 A 股、美股/美股指数、港股指数、内外盘期货和汇率品种。
- 支持删除预设行情与恢复默认；显示新品种时会立即补取行情与可用的当天趋势数据。
- 迷你趋势线按各市场的实际日内时间进度绘制，并以相对昨收的涨跌方向显示颜色。
- 默认国内金品种调整为沪金 99（`SGE_AU9999`）。
- 主窗与背景层作为 Windows 工具窗口运行，不单独占用任务栏；改进双屏负坐标保存及屏幕布局变化后的可见位置恢复。

## V1.1.0 - 2026-05-27

- 行情源统一为新浪财经，并加入迷你趋势线及趋势透明度设置。
- 新增文字透明度设置，趋势线可为未发生的交易时段保留右侧空白。

## V1.0.0 - 2026-05-26

- 首次发布 Windows 单文件桌面行情小组件。
