# 桌面实时行情

一个可打包成 Windows 免安装程序的桌面行情窗，用来显示：

- 纽约金现价
- 国内金价 Au(T+D)
- 纳指期货价格和涨跌幅
- 上证指数、科创 50
- 纽约原油
- 美元兑人民币、日元兑人民币

## 推荐使用方式

直接双击：

```text
dist\桌面实时行情.exe
```

这是单文件程序，可以复制到其他 Windows 电脑上直接运行。

## Python 版本开发运行

```powershell
python desktop_ticker.py
```

## 生成单文件 exe

```powershell
.\build-python-exe.ps1
```

生成文件在 `dist\桌面实时行情.exe`。

## Electron 版本开发运行

```powershell
npm install
npm start
```

## Electron 版本打包

```powershell
npm run dist
```

生成文件在 `release/` 目录中。注意：`release\win-unpacked\桌面实时行情.exe` 必须和同目录的 `resources`、`locales`、DLL 文件一起使用，不能只复制 exe。

也可以直接运行：

```powershell
.\build-windows.ps1
```

说明：一个“所有系统都通用”的单文件程序在桌面软件里并不现实。当前项目面向 Windows 便携版；如果要给 macOS 或 Linux 用户使用，需要在对应系统上分别打包。

## 使用说明

- 窗口默认置顶、无边框、背景低透明，可拖动。
- 双击小组件手动刷新。
- 右键小组件退出。
- 顶部 `↻` 刷新，`⌖` 切换置顶，`⚙` 打开设置，`🔒` 可防止误触和拖动，`×` 退出。
- 点击 `🔒` 后会锁住窗口位置和大小，顶部按键仍可操作；自动行情刷新仍会继续。
- 关闭程序时会记住当前窗口位置，下次启动会恢复到上次位置。
- 程序只允许打开一个窗口；已有窗口运行时再次启动不会重复打开。
- 设置窗口分为行情、显示、颜色、行为四页；可勾选要显示的行情，并按住右侧三横杠拖动排序。
- 设置的“行情”页可输入多个自选 A 股代码，例如 `600519, 000001, 300750`；添加后输入框会清空，股票会保存在上方列表，可直接删除。
- 设置里也可调整背景颜色、背景透明度、文字颜色、价格字号、小数位数、涨跌颜色、刷新间隔、默认置顶；颜色可设为“无”。
- 窗口大小可拖动主窗口右下角调整，会自动保存。
- 程序默认每 30 秒自动刷新一次，可在设置中修改。

## 行情源说明

默认行情源：

- 纽约金：Yahoo Finance `GC=F`
- 国内金价：新浪财经 `SGE_AUTD`
- 纳指期货：Yahoo Finance `NQ=F`
- 上证指数：新浪财经 `sh000001`
- 科创 50：新浪财经 `sh000688`
- 纽约原油：Yahoo Finance `CL=F`
- 美元兑人民币：Yahoo Finance `CNY=X`
- 日元兑人民币：Yahoo Finance `JPYCNY=X`

这些都是第三方公开接口，可能会因为网络、地区、反爬或接口变更而不可用。Python 单文件版的行情获取逻辑集中在 `desktop_ticker.py`，Electron 版的行情获取逻辑集中在 `src/main.js`。
