# 桌面实时行情

一个可打包成 Windows 免安装程序的桌面行情窗，用来显示：

- 纽约金现价
- 沪金 99
- 纳指期货价格和涨跌幅
- 上证指数、科创 50
- 纽约原油
- 美元兑人民币、日元兑人民币

## 推荐使用方式

直接双击：

```text
dist\桌面实时行情.exe
```

这是单文件程序，可以复制到其他 Windows 电脑上直接运行。

已发布的安装包可从 [GitHub Releases](https://github.com/thebsf/StockPopup/releases) 下载；各版本变化见 [CHANGELOG.md](CHANGELOG.md)。

## Python 版本开发运行

```powershell
python desktop_ticker.py
```

## 生成单文件 exe

```powershell
.\build-python-exe.ps1
```

生成文件在 `dist\桌面实时行情.exe`。

## Electron 版本开发运行

```powershell
npm install
npm start
```

## Electron 版本打包

```powershell
npm run dist
```

生成文件在 `release/` 目录中。注意：`release\win-unpacked\桌面实时行情.exe` 必须和同目录的 `resources`、`locales`、DLL 文件一起使用，不能只复制 exe。

也可以直接运行：

```powershell
.\build-windows.ps1
```

说明：一个“所有系统都通用”的单文件程序在桌面软件里并不现实。当前项目面向 Windows 便携版；如果要给 macOS 或 Linux 用户使用，需要在对应系统上分别打包。

## 使用说明

- 窗口默认置顶、无边框、背景低透明，可拖动。
- 双击小组件手动刷新。
- 右键小组件退出。
- 顶部 `↻` 刷新，`⌖` 切换置顶，`⚙` 打开设置，`🔒` 可防止误触和拖动，`×` 退出。
- 点击 `🔒` 后会锁住窗口位置和大小，顶部按键仍可操作；自动行情刷新仍会继续。
- 关闭程序时会记住当前窗口位置，下次启动会恢复到上次位置。
- 程序只允许打开一个窗口；已有窗口运行时再次启动不会重复打开。
- 主窗口与半透明背景层按 Windows 工具窗口处理，不在任务栏单独显示；在双屏或副屏位于主屏左侧/上方时会正确保留负坐标位置，并在屏幕布局变更后把不可见窗口限制回可见区域。
- 每条行情会在名称与价格之间显示当天的迷你趋势线；曲线颜色与该品种相对昨收的当日涨跌颜色一致。A 股/指数按当日交易时段绘制，外盘期货按新浪分时时间轴绘制，汇率按当天 24 小时时间轴绘制，未走完的时间右侧会保留空白；缺少可解析分时数据的品种只累计程序运行期间的走势。
- 设置窗口分为行情、显示、颜色、行为四页；可勾选要显示的行情，并按住右侧三横杠拖动排序。
- 设置的“行情”页可输入新浪 App 里看到的代码或名称，程序会尝试自动解析成实际行情符号。例如 `NKY`、`HSKTECH`/`HSTECH`、`日经225指数期货`、`恒生科技指数`、`纳斯达克`、`纽约黄金`、`600519`、`US.AAPL`、`SGE.AU9999`。添加后可直接删除。
- 预设行情与自选行情均可删除；删除预设后可使用设置页的“恢复默认”找回。当没有任何显示行情时，主窗口会提示在设置中添加。
- 新勾选显示的品种会立即刷新行情与可用的当天趋势数据；如果正好有一轮刷新进行中，也会在该轮完成后立刻补取，不需要等待下一轮自动刷新。
- 已验证的自动转换包括：`NKY` -> 日经225指数，`HSKTECH` -> 恒生科技指数，`US.IXIC`/`纳斯达克` -> 纳斯达克指数，`日经225指数期货` -> 日经225指数期货。新浪不同市场使用不同后台符号，无法解析的代码会在行情列表中显示失败，便于继续补充对应规则。
- 系统会先按代码所属市场自动尝试新浪数值趋势接口；能取得分时数据就按真实时间位置绘制，接口无数据时才累计程序运行期间的走势。当前已确认数值日趋势的类别包括 A 股/内地指数、`US.*` 美股/美股指数、`HF.*` 外盘期货、`NF.*` 国内期货、`FX.*` 汇率、`HSTECH` 恒生科技指数与 `NKY` 日经225指数。`SGE.AU9999` 沪金99 实时报价可用，但已探测的新浪分时端点仍返回空数据，因此当前保留本地累计走势。
- 设置里也可调整背景颜色、背景透明度、文字透明度、趋势线透明度、文字颜色、价格字号、小数位数、涨跌颜色、刷新间隔、默认置顶；颜色可设为“无”。
- 窗口大小可拖动主窗口右下角调整，会自动保存。
- 程序默认每 30 秒自动刷新一次，可在设置中修改。

## 行情源说明

默认行情源：

- 纽约金：新浪财经 `hf_GC`
- 沪金 99：新浪财经 `SGE_AU9999`（可输入简写 `AU9999`）
- 纳指期货：新浪财经 `hf_NQ`
- 上证指数：新浪财经 `sh000001`
- 科创 50：新浪财经 `sh000688`
- 纽约原油：新浪财经 `hf_CL`
- 美元兑人民币：新浪财经 `fx_susdcny`
- 日元兑人民币：新浪财经 `fx_sjpycny`

这些都是第三方公开接口，可能会因为网络、地区、反爬或接口变更而不可用。Python 单文件版的行情获取逻辑集中在 `desktop_ticker.py`，Electron 版的行情获取逻辑集中在 `src/main.js`。
